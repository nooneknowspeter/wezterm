## setup

### linux / macos

```sh
rm -rf ~/.config/wezterm
git clone https://github.com/nooneknowspeter/wezterm.git ~/.config/wezterm
```

### windows

```powershell
rm -f $HOME\.config\wezterm
git clone https://github.com/nooneknowspeter/wezterm.git $HOME\.config\wezterm
```
